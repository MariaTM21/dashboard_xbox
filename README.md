# dashboard_xbox
Foi utilizado a base de dados disponibilizada no curso da DIO, que nela contém dados do XBOX GAME PASS.
Foi realizada uma análise da base, feito ajustes em alguns tipos de dados, logo em seguida criado um padrão de abas para melhor gerenciamento das informações. Optou se por fazer alguns gráficos, realizadas algumas perguntas de negócios para melhor apresentação.
